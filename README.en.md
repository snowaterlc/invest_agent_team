# Invest Agent Team Project

This project is an open-source initiative developed by the [Invest Agent Team](https://gitee.com/snowater/invest_agent_team), designed to provide functionalities and services related to investment agency.

## Project Overview

This project primarily focuses on development within the investment agency domain, with specific features and objectives including:
- Implementing business logic for investment agency services.
- Supporting agency services across multiple investment scenarios.
- Providing an extensible architecture design to facilitate future feature expansion and integration.

## Features

- **Investment Agency Service**: Core implementation of investment agency business logic.
- **Multi-Scenario Support**: Designed to support agency services in various investment scenarios.
- **Extensibility**: Modular architecture for easy future feature expansion.
